# Codex Local Session Manager

[English](./README.md) | [Simplified Chinese](./README.zh-CN.md)

A local web tool for browsing and cleaning Codex session history.

## Quick Start

1. Open `index.html` in a Chromium-based browser.
2. Click `Pick directory`.
3. Authorize your `.codex` directory and start managing local sessions.

## Key Features

- Single-file `index.html`
- Fully local, no backend required
- Pagination, search, and filtering for session history
- Full conversation preview for each session
- Delete current, selected, or all sessions
