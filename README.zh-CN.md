# Codex Local Session Manager

[English](./README.md) | [简体中文](./README.zh-CN.md)

一个纯本地的 Codex 历史会话管理工具.

## 快速开始

1. 使用 Chromium 内核浏览器打开 `index.html`.
2. 点击 `选择目录`.
3. 授权你的 `.codex` 目录后开始管理本地会话.

## 核心特性

- 单文件 `index.html`
- 纯本地运行, 无后端依赖
- 支持会话历史的分页, 搜索和筛选
- 支持查看每个会话的完整对话内容
- 支持删除当前, 选中或全部会话
